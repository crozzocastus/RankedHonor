# Auth Routes Group / Grupo de Rotas de Autenticação

## English

This route group contains all authentication-related pages (login, register).

### Purpose

- Groups authentication pages together
- Applies shared authentication layout
- The `(auth)` folder name doesn't appear in URLs
- Provides consistent UI/UX for auth flows

### Pages

- **login/** - User login page
- **register/** - New user registration page

### Features

- Shared layout for consistent auth experience
- Form validation and error handling
- Redirect to dashboard after successful authentication
- Link between login/register pages

### URL Structure

- `/login` - Login page
- `/register` - Registration page

Note: The `(auth)` folder name is not part of the URL

## Português

Este grupo de rotas contém todas as páginas relacionadas à autenticação (login, registro).

### Propósito

- Agrupa páginas de autenticação juntas
- Aplica layout compartilhado de autenticação
- O nome da pasta `(auth)` não aparece nas URLs
- Fornece UI/UX consistente para fluxos de autenticação

### Páginas

- **login/** - Página de login do usuário
- **register/** - Página de registro de novo usuário

### Funcionalidades

- Layout compartilhado para experiência consistente de autenticação
- Validação de formulário e tratamento de erros
- Redirecionamento para dashboard após autenticação bem-sucedida
- Link entre páginas de login/registro

### Estrutura de URLs

- `/login` - Página de login
- `/register` - Página de registro

Nota: O nome da pasta `(auth)` não faz parte da URL
